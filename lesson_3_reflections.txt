Describe the differences between forks, clones, and branches. When would you use one instead of another?

Fork is to break off your own version from a git repository if you can't edit the original. There won't be a merge of the two.

Clones is making a local copy of the GitHub repository or vice versa

Branches are different code paths on the same respository
